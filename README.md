# Hockey Nation OVR Calculator

Calculates min & max potential OVR for players in [hockey-nation](https://hockey-nation.com).

## Install

- **[Chrome Web Store](https://chromewebstore.google.com/detail/hockey-nation-ovr-calcula/hnbigibapnnbbponinblinhbaomblmlb)** (recommended)
- Or grab the latest zip from [Releases](../../releases), unzip, go to `chrome://extensions`, enable developer mode, click **Load unpacked**, select the folder.

## Build

```bash
npm install
npm run build
```
